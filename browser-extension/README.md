# YouTube 재생목록 추출기 - 브라우저 확장 프로그램

Watch Later 재생목록을 추출하기 위한 Chrome/Edge 확장 프로그램입니다.

## 기능

- ✅ YouTube Watch Later 페이지에서 영상 정보 추출
- ✅ JSON, Markdown, HTML 형식으로 다운로드
- ✅ Python 스크립트와 동일한 출력 형식 지원
- ✅ 로컬에서만 동작 (데이터 전송 없음)

## 설치 방법

1. Chrome/Edge 브라우저에서 `chrome://extensions/` 접속
2. "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 이 디렉토리 선택

## 사용 방법

1. YouTube에서 Watch Later 페이지 열기: `https://www.youtube.com/playlist?list=WL`
2. 확장 프로그램 아이콘 클릭
3. "추출 시작" 버튼 클릭
4. 자동으로 스크롤하며 모든 영상 수집
5. 완료 후 다운로드 버튼 클릭
